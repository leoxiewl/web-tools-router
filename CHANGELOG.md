# Changelog

All notable changes to this project will be documented here.

## [0.1.0] - 2026-07-17

### Added

- Cross-agent Web task routing Skill.
- Separate routes for keyword search and known-URL retrieval.
- Deterministic OpenCLI adapter selection before browser automation where appropriate.
- agent-browser fallback for rendered, authenticated, and interactive pages.
- Safety rules for external side effects, authentication, downloads, and untrusted content.
- Routing evaluation cases for future regression checks.
